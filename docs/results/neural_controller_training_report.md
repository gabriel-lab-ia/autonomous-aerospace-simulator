This report records a real execution of:

```bash
uv run python scripts/train_neural_controller.py
```

The run performs supervised neural-controller pretraining on synthetic
telemetry generated by the rule-based teacher in the training script.

## Interpretation And Limitations

These metrics measure agreement with synthetic labels, not successful rocket
landing. The model has not been validated against high-fidelity aerospace
dynamics, hardware, disturbances, or real flight data. Simulator-loop
inference is integrated as an experimental neural control layer and provides a
foundation for future simulator-in-the-loop reinforcement learning and
controller benchmarking.
